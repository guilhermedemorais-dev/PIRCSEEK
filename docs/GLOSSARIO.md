# Glossario: PIRCSEEK

## PIRCSEEK

Buscador local de contexto para CLIs de IA.

## PIRC

Metodologia de recuperacao progressiva de contexto.

## SEEK

Do ingles, procurar ou buscar.

## Categoria

Agrupamento configuravel de itens, como skills, MCPs, memoria, projetos ou
prompts.

## Item

Unidade cadastrada no PIRCSEEK. Possui titulo, resumo, tags, Markdown completo,
tokens estimados e linha JSONL.

## Subindice

Indice JSONL especifico de uma categoria.

## JSONL

Formato com um JSON por linha. Usado para LLM consumir metadados com baixo custo
de tokens.

## Markdown

Formato humano para conteudo completo.

## Doctor

Comando de diagnostico que verifica se o PIRCSEEK esta instalado e saudavel.

## SDD

Specification Driven Development. O codigo deve seguir uma especificacao clara
antes de expandir escopo.

## TDD

Test Driven Development. Os testes guiam a implementacao e validam comportamento.
